# Expense Tracker Web Application

## Project Overview
The **Expense Tracker** is a web application developed using **React JS, HTML, CSS, and JavaScript**.  
It helps users track their daily expenses, view total spending, and calculate the remaining balance based on a predefined budget.  

This project is perfect for learning **React fundamentals**, state management, and dynamic UI updates.

---

## Features
- Add a new expense (expense name + amount)  
- Delete an expense  
- View total expenses dynamically  
- Display remaining balance (Budget âˆ’ Total Expense)  
- Responsive and user-friendly interface

## Technologies Used
- **Frontend:** HTML, CSS  
- **Programming Language:** JavaScript  
- **Framework:** React JS (Functional Components + useState)  
- **Data Storage:** Browser LocalStorage (optional for persistence)

## Demo
1. On page load, the app shows:
   - Total budget  
   - Total expense  
   - Remaining balance  

2. Users can:
   - Enter expense name and amount  
   - Click **Add Expense** to save  
   - Delete expenses anytime with a **Delete button**  

3. The UI updates automatically using React state.
